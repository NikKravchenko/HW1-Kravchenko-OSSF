# Домашнє завдання №3: управління процесами та моніторинг ресурсів

## Завдання 1. Огляд активних процесів

1. Виведення списку всіх процесів у системі.

   ```bash
   ps aux
   ```

2. Запуск інтерактивного монітора для пошуку процесу з найбільшим споживанням RAM.

   ```bash
   top
   ```

3. Визначення PID процесу поточної оболонки.

   ```bash
   echo $$
   ```

## Завдання 2. Робота у фоні та керування процесами

1. Запуск довгої команди у фоновому режимі.

   ```bash
   sleep 1000 &
   ```

2. Перевірка списку фонових завдань.

   ```bash
   jobs
   ```

3. Повернення процесу з фону на передній план.

   ```bash
   fg %1
   ```

4. Примусове завершення процесу після його зупинки (через Ctrl + Z).

   ```bash
   kill -9 %1
   ```

5. Запуск команди через `nohup` для роботи після закриття терміналу.

   ```bash
   nohup sleep 2000 &
   ```

## Завдання 3. Пріоритети та обмеження

1. Запуск команди з підвищеним значенням `nice` (зниженим пріоритетом).

   ```bash
   nice -n 10 sleep 500 &
   ```

2. Зміна пріоритету вже запущеного процесу за його PID.

   ```bash
   renice -n 15 -p 1234
   ```

3. Перегляд поточних обмежень ресурсів користувача.

   ```bash
   ulimit -a
   ```

## Завдання 4. Моніторинг ресурсів

1. Виведення інформації про використання дискового простору.

   ```bash
   df -h
   ```

2. Показати обсяг оперативної пам'яті у зручному форматі.

   ```bash
   free -h
   ```
