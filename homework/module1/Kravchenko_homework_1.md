# Домашнє завдання №1 з основ Linux

## Завдання 1

1. `ls -la` — показати вміст поточної директорії, включно з прихованими файлами.
2. `cd ~/Downloads` — перейти до папки Downloads.
3. `ls -la` — показати вміст каталогу Downloads.
4. `> test.txt` — створити порожній файл `test.txt` без використання `touch`.
5. `cat test.txt` — вивести вміст файлу `test.txt` (у цьому випадку файл порожній).
6. `cd /Users/nik` — перейти в домашній каталог за абсолютним шляхом.
7. `cd ..` — перейти в домашній каталог за відносним шляхом (наприклад, із `/Users/nik/Downloads` до `/Users/nik`).

## Завдання 2

Виконані команди:

1. `man ls`
2. `help cd`
3. `man cat`
4. `man man`
5. `cp --help`
6. `mv --help`

- Ключ `ls` для показу прихованих файлів: `-a`.
- Ключ `cat` для нумерації рядків: `-n`.
- Відмінність `man` і `--help`:
  - `man` відкриває повне керівництво з навігацією за розділами.
  - `--help` виводить коротку довідку прямо в термінал.

## Завдання 3

Мінісценарій із 8 команд:

1. `cd ~/Downloads` — перейти в каталог Downloads.
2. `mkdir linux_hw` — створити каталог `linux_hw`.
3. `cd linux_hw` — перейти в створений каталог.
4. `> note.txt` — створити один порожній файл `note.txt`.
5. `ls -la` — переглянути список файлів у поточному каталозі.
6. `cd /tmp` — перейти в інше місце файлової системи.
7. `cd ~/Downloads/linux_hw` — повернутися назад у робочий каталог.
8. `man ls` — переглянути документацію команди `ls`.
