# Домашнє завдання №1 з основ Linux

## Завдання 1

1. `ls -la` — показати вміст поточної директорії, включно з прихованими файлами.
2. `cd ~/Downloads` — перейти до папки Downloads.
3. `> test.txt` — створити порожній файл `test.txt` без використання `touch`.
4. `cat test.txt` — вивести вміст файлу `test.txt` (у цьому випадку файл порожній).
5. `cd /Users/username` — приклад переходу за абсолютним шляхом до домашньої директорії користувача.
6. `cd ~` — перехід до домашньої директорії за відносним/скороченим шляхом.

## Завдання 2

- Ключ `ls` для показу прихованих файлів: `-a`.
- Ключ `cat` для нумерації рядків: `-n`.
- Відмінність `man` і `--help`:
  - `man` відкриває повне керівництво з навігацією за розділами.
  - `--help` виводить коротку довідку прямо в термінал.

## Завдання 3

Мінісценарій із 6 команд:

1. `cd ~/Downloads` — перейти до робочої папки.
2. `mkdir linux_hw` — створити нову директорію `linux_hw`.
3. `echo "Hello, Linux!" > linux_hw/note.txt` — створити файл `note.txt` і записати в нього текст.
4. `ls -la linux_hw` — подивитися вміст створеної директорії.
5. `cd ..` — піднятися на один рівень вище.
6. `man ls` — відкрити повне керівництво з команди `ls`.
