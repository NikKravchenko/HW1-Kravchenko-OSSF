# Домашнє завдання №2: файлова система і права доступу

## Завдання 1. Ієрархія каталогів Linux

1. Перехід у кореневий каталог `/` і показ вмісту.

   ```bash
   cd /
   ls -la
   ```

2. Перехід у `/etc` і показ вмісту.

   ```bash
   cd /etc
   ls -la
   ```

3. Перехід у `/home` і показ списку користувачів (домашніх каталогів).

   ```bash
   cd /home
   ls -la
   ```

## Завдання 2. Файли, каталоги та посилання

1. Створення нового каталогу в домашньому каталозі.

   ```bash
   mkdir ~/lab2
   ```

2. Створення файлу всередині каталогу.

   ```bash
   touch ~/lab2/file.txt
   ```

3. Копіювання файлу в нове ім’я.

   ```bash
   cp ~/lab2/file.txt ~/lab2/file_copy.txt
   ```

4. Перейменування копії.

   ```bash
   mv ~/lab2/file_copy.txt ~/lab2/file_renamed.txt
   ```

5. Створення жорсткого посилання.

   ```bash
   ln ~/lab2/file.txt ~/lab2/file_hard.txt
   ```

6. Створення символічного посилання.

   ```bash
   ln -s ~/lab2/file.txt ~/lab2/file_sym.txt
   ```

7. Пошук файлу за ім’ям у каталозі `/home`.

   ```bash
   find /home -name file.txt
   ```

## Завдання 3. Права доступу

1. Перегляд прав доступу створеного файлу.

   ```bash
   ls -l ~/lab2/file.txt
   ```

2. Надання файлу прав лише на читання для всіх (власник/група/інші).

   ```bash
   chmod 444 ~/lab2/file.txt
   ```

3. Надання власнику права на запис.

   ```bash
   chmod u+w ~/lab2/file.txt
   ```

4. Перегляд поточного значення `umask`.

   ```bash
   umask
   ```

5. Встановлення нового значення `umask`.

   ```bash
   umask 022
   ```

## Завдання 4. Користувачі

1. Створення нового користувача (потрібні права адміністратора).

   ```bash
   sudo useradd -m trainee
   ```

2. Додавання користувача до групи `sudo`.

   ```bash
   sudo usermod -aG sudo trainee
   ```

3. Перевірка, що користувач існує в системі.

   ```bash
   grep trainee /etc/passwd
   ```
